Pierakład paasobnych fraz

  1. U sučasnych hartačoch isnuje mahčymaść dadać pošuk (pravier prapanavananyja pierakłady)
  2. Nie sakret, što akno pošuku prosta ŭvodzić adras URL u vyhladzie http://be-latin.open-tran.eu/suggest/search engine.
  3. Karystalniki Firefoksa mohuć dadać zakładku z klučavym słovam. Kab heta zrabić, uvachodzim u nałady zakładak i dadajem novuju takim čynam:



    Ad hetaj pary pry kožnym napisańni tran fraza u poli URL, aŭtamatyčna ŭklučycca pošuk.

XML RPC

Servisam možna karystacca pry dapamozie luboj prahramy, zdolnaj vykanać addaleny zahad praz pratakoł XML RPC. Z hetaj metaj treba skarystać metad suggest, jaki apracoŭvaje dva arhumenty: pieršy - heta fraza, jakuju treba pierakłaści, a druhi - heta kod movy, na jakuju maje adbycca pierakład (naprykład "be_latin").

U Pythonie heta robicca naprykład tak: 

 from xmlrpclib import ServerProxy

 lang = "be-latn" # zamianić na kod vašaj movy
 server = ServerProxy("http://open-tran.eu/RPC2")
 for sugg in server.suggest("save as", lang):
 print sugg

Nieŭzabavie pastarajusia źmiaścić prosty redaktar fajłaŭ .po, jaki budzie pry dapamozie takoha mechanizmu šukać padkazki pry pierakładach paasobnych fraz.
