    Òd terô pòwtôrzné kòżdą razą wpisanié adresë w sztaturze tran fraza sprawi aùtomatné wëwòłanié szëkbë.

XML RPC

Z serwisu mòże brëkòwac równojaką programą, co rozmie robic daleczé wëwòłanié przez protokół XML RPC. Nót je w tim célu zwënégòwac metodã suggest, jaka pòbiérô dwa argùmentë: pierszi to fraza dlô dolmaczënkù, a drëdżi to dwa- abò trzëlëterowi kòd jãzëka, na jaczi mô òstac przedolmaczonô.

W Pythonie robi sã to na przëmiôr tak: 

      from xmlrpclib import ServerProxy

      server = ServerProxy("http://open-tran.eu/RPC2")
      for sugg in server.suggest("save as", "csb"):
          print sugg

Wnetka dóm prosti editork lopków .po, jaczi mdze brëkùjąc nen werk pòbiérôł pòdpòwiescë dolmaczënków apartnëch frazów.
