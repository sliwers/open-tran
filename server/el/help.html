XML RPC

Μπορείτε να χρησιμοποιήσετε αυτήν την υπηρεσία με οποιοδήποτε πρόγραμμα που έχει τη δυνατότητα να καλέσει μια απομακρυσμένη διαδικασία μέσω του πρωτοκόλλου XML RPC. Πρέπει να χρησιμοποιήσετε τη μέθοδο suggest, που δέχεται δύο παραμέτρους: μια φράση προς μετάφραση και έναν κωδικό γλώσσας προορισμού.

Στην Python, μπορείτε να το χρησιμοποιήσετε με τον ακόλουθο τρόπο: 

      from xmlrpclib import ServerProxy

      lang = "el" # αντικαταστήστε το με τον κωδικό γλώσσας που θέλετε
      server = ServerProxy("http://open-tran.eu/RPC2")
      for sugg in server.suggest("save as", lang):
          print sugg

Σύντομα, Θα προσπαθήσω να ανεβάσω έναν απλό επεξεργαστή αρχείων .po που θα μπορεί να πάρει προτάσεις μετάφρασης από τον εξυπηρετητή με τη χρήση αυτού του μηχανισμού.
