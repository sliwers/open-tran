Traducción de las frases

  1. En todos los navegadores modernos se puede añadir un search engine (véate las sugerencias de traducción).
  2. Por supuesto, no es un secreto que la ventana de búsqueda introduzca una dirección URL de modo siguiente: http://es.open-tran.eu/suggest/search engine.
  3. Los usuarios de Firefox pueden crear en su navegador un marcador con palabra clave. Para hacer esto, vamos a 'Marcadores -> Administrar marcadores' y eligemos la opción Nuevo marcador:



    A partir de ahora, cuando tecleemos tran frase en la barra de direcciones, empecerá la traducción.

XML RPC

Para poder utilizar este servicio se necesita un programa que permite realizar las llamadas a procedimientos remotos a través del protocolo XML RPC. Para ello hay que emplear el método suggest que toma dos argumentos: la frase para traducir y el código de dos letras del idioma al que se traduce (o sea "es" en el caso del español).

Por ejemplo en Python se lo hace así: 

      from xmlrpclib import ServerProxy

      server = ServerProxy("http://open-tran.eu/RPC2")
      for sugg in server.suggest("save as", "es"):
          print sugg
