Yksittäisten fraasien kääntäminen

  1. Voit lisätä hakukoneen (katso ehdotetut käännökset) mihin tahansa nykyaikaiseen webselaimeen.
  2. Ei ole salaisuus, että hakusivu menee seuraavanlaiseen osoitteeseen: http://fi.open-tran.eu/suggest/search engine.
  3. Firefoxin käyttäjät voivat lisätä kirjanmerkin avainsanalla varustettuna avaamalla Kirjanmerkit-valikosta Muokkaa kirjanmerkkejä -ikkunan ja lisäämällä uuden kirjanmerkin seuraavasti:



    Voit sitten hakea fraasia kirjoittamalla osoiteriville tran fraasi.

XML RPC

Palvelua voi käyttää millä tahansa ohjelmalla, joka pystyy suorittamaan etäkutsun XML RPC-yhteyskäytännöllä. Sinun tulisi käyttää suggest-komentoa, jolla on 2 parametria: käännettävä fraasi ja kohdekielen koodi.

Python-kielellä tehtynä esimerkiksi seuraavasti: 

      from xmlrpclib import ServerProxy

      server = ServerProxy("http://open-tran.eu/RPC2")
      for sugg in server.suggest("save as", "fi"):
          print sugg

Yritän lähettää pian yksinkertaisen .po-muokkaimen, joka osaa noutaa ehdotukset palvelimelta tällä tapaa.
