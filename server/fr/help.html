Traduction de simples phrases

  1. Vous pouvez ajouter un search engine (regardez les suggestions de traduction pour ce mot) dans tous les navigateurs modernes.
  2. Tout le monde peut voir que le formulaire de recherche utilise une adresse (URL) du format suivant : http://fr.open-tran.eu/suggest/search engine.
  3. Les utilisateurs de Firefox peuvent ajouter un marque-page avec un mot-clé. Pour cela il suffit de cliquer sur "Organiser les marque-pages" dans le menu "Marque-pages" et d'en ajouter un nouveau comme ceci :



    Et maintenant, entrer une adresse du format trad phrase lancera automatiquement une recherche.

XML RPC

Vous pouvez utiliser ce service avec n'importe quel programme capable d'appeler une procédure via le protocole XML RPC. Vous devriez employer la méthode suggest, qui utilise deux arguments : la phrase à traduire et le code de la langue de destination.

En Python, vous pouvez l'utiliser de cette façon : 

      from xmlrpclib import ServerProxy

      server = ServerProxy("http://open-tran.eu/RPC2")
      for sugg in server.suggest("save as", "fr"):
          print sugg

J'essaierai de mettre bientôt en ligne un simple éditeur de .po qui pourra chercher les suggestions sur le serveur en utilisant ce mécanisme.
