It oersetten fan losse sinnen

  1. Jo kinne in sykmasine (kar út foarstelde oersettings) oan eltse moderne webblêder taheakje.
  2. It sykformulier wurdt ynfierd fia in URL mei it folgjende formaat: http://fy.open-tran.eu/suggest/search engine.
  3. Firefox-brûkers kinne in blêdwizer mei in kaaiwurd taheakje. Dêrfoar iepenje jo it finster "Organize Bookmarks" fan it menu "Bookmarks" en heakje de blêdwizer sa ta:



    As jo no it formulier tran phrase fia de URL ynfiere wurdt de sykfraach automatysk oanbean.

XML RPC

Jo kinne dizze tsjinst brûke by alle programma's dy't ynsteat binne in remote-proseduere op te roppen fia it XML RPC foarskrift. Jo moatte de metoade suggest brûke, dy't twa arguminten freget: in sin om oer te setten en de winske taalkoade.

Yn Python kinne jo it sa brûke: 

      from xmlrpclib import ServerProxy

      server = ServerProxy("http://open-tran.eu/RPC2")
      for sugg in server.suggest("save as", "fy"):
          print sugg

Ik sil besykje gau in ienfâldige po-tekstbewurker op it Web te laden dy't ynsteat is om fia sa'n mechanisme suggestjes fan de tsjinner te heljen.
