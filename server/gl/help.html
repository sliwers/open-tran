Tradución de frases soltas

  1. Pode engadir un motor de procuras (consulte as suxestións de tradución) en calquer navegador moderno.
  2. Non é nengún segredo que o formulario de procura dirixe a un URL co seguinte formato: http://gl.open-tran.eu/suggest/search engine.
  3. Os usuarios de Firefox poden engadir un marcador cunha palabra-chave. Para facer isto debe abrir a fiestra "Organizar Marcadores" no menú "Marcadores" e engadir un novo do seguinte xeito:



    Agora, se escrebe un URL da forma tran frase fará automaticamente unha pesquisa.

XML RPC

Pode usar este servizo con calquer programa que sexa capaz de activar un procedimento remoto mediante o protocolo XML RPC. Debe usar o método suggest, que toma dous argumentos: unha frase a traducir e o código da lingua de destino.

Con Python, pode usálo do seguinte xeito: 

      from xmlrpclib import ServerProxy

      lang = "gl" # substitúao co código do seu idioma
      server = ServerProxy("http://open-tran.eu/RPC2")
      for sugg in server.suggest("save as", lang):
          print sugg
