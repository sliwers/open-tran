XML RPC

תוכל להשתמש בשירות זה עם כל תוכנית המסוגלת לקרוא להליך מרוחק דרך פרוטוקול ה- XML RPC. עליך להשתמש בשיטת ההצעה, הלוקחת שני משתנים: מונח לתרגום ואת קוד שפת היעד.

ב-Python, תוכל להשתמש בדרך הבאה: 

      from xmlrpclib import ServerProxy

      lang = "he" # replace with your language code
      server = ServerProxy("http://open-tran.eu/RPC2")
      for sugg in server.suggest("save as", lang):
          print sugg

אני אנסה להעלות עורך .po פשוט ברגע שאוכל לקבל הצעות מהשרת באמצעות מנגנון זה.
