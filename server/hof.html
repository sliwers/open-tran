Software

  1. Translate Toolkit
    Translate Toolkit provides complete support for many file formats. It is used to import several gigabytes of translations into a single database.
  2. Natural Language Toolkit
    Open-Tran uses word stemmers from NLTK.
  3. Ubuntu GNU/Linux
  4. Python
    The server, as well as the importing scripts are developed in Python.
  5. SQLite
    More than 50MM words in about 10MM phrases stored in a multi-gigabyte database. SQLite handles it on a virtual machine with only 128MB of RAM.
  6. Emacs
    The ultimate development environment
