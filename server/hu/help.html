XML RPC

Használhatja ezt a szolgáltatást bármilyen alkalmazással, amely képes távoli eljárások hívására a XML RPC protokol segítségével. Használja a suggest módszert, amely két paramétert tartalmaz: a fordítani kívánt szót és a célnyelv kódját.

Python esetében használja a következő képpen: 

      from xmlrpclib import ServerProxy

      lang = "pt_BR" # írja felül a saját nyelvének megfelelő kóddal
      server = ServerProxy("http://open-tran.eu/RPC2")
      for sugg in server.suggest("save as", lang):
          print sugg

A közeljövőben feltöltünk egy egyszerű .po szerkesztőt, ami lehetővé teszi a javaslatok átvételét a szerverről az előző módszer segítségével.
