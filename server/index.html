"Open" what?

This service offers an easy access to the translations of open source projects. It acts as a kind of a central Memory Translation database where you can look up a translation suggestion or submit a whole .po file to the server. Just go ahead: find your language below and try it yourself.

Who pays for this?

This service should cost me about 90€ until February, 2008. As long as I have a decently paid job, I can afford it. I have been using open source software for many years now and I believe I owe something to the community. Let it be my tribute.
