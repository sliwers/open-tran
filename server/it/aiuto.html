Traduzione di frasi singole

  1. Puoi aggiungere un motore di ricerca, in inglese si dice "search engine" (vedi la traduzione suggerita), a qualsiasi browser web moderno.
  2. Naturalmente non è un segreto che il modulo di ricerca utilizzi un URL nel seguente formato: http://it.open-tran.eu/suggest/search engine.
  3. Se sei un utente di Firefox poi aggiungere un segnalibro con parola chiave. Per poterlo fare devi aprire la finestra "Gestione segnalibri..." dal menu "Segnalibri" e premere il pulsante "Nuovo segnalibro". Poi compila i campi come segue:

XML RPC

Puoi usare questo servizio con qualsiasi programma che sia in grado di effettuare una chiamata a procedura remota attraverso il protocollo XML RPC. Dovresti utilizzare il metodo suggest, che prende due argomenti: una frase da tradurre ed il codice di due lettere della lingua nella quale vuoi tradurre (ovvero "it" nel caso dell'italiano).

In Python, ad esempio, si può fare in questo modo: 

      from xmlrpclib import ServerProxy

      server = ServerProxy("http://open-tran.eu/RPC2")
      for sugg in server.suggest("save as", "it"):
          print sugg

Proverò ad inviare prossimamente un semplice editor di file .po che sia in grado di ottenere i suggerimenti di traduzione dal server attraverso questo meccanismo.
