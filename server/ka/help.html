XML RPC

ამ სერვისის გამოყენება შესაძლებელია ყველა იმ პროგრამასთან, რომელსაც XML RPC პროტოკოლის მეშვეობით დაშორებული პროცედურების გამოძახება შეუძლია. გამოიყენეთ მეთოდი suggest რომელიც იღებს ორ არგუმენტს: სათარგმნ წინადადებას და ენის კოდს.

პითონში ყოველივე ამის გამოყენება ხდება ამდაგვარად: 

      from xmlrpclib import ServerProxy

      lang = "pt_BR" # replace with your language code
      server = ServerProxy("http://open-tran.eu/RPC2")
      for sugg in server.suggest("save as", lang):
          print sugg
