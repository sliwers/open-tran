XML RPC

XML RPC 프로토콜을 이용하여 원격 프로시저를 사용할 수 있는 모든 플랫폼에서 동일한 기능을 사용할 수 있습니다. suggest 메서드를 사용해야 하며, 두 개의 인수(번역할 단어와 대상 언어)가 필요합니다.

파이썬에서는 다음과 같이 사용합니다. 

      from xmlrpclib import ServerProxy

      lang = "ko_KR" # 원하는 언어 코드로 변경하십시오.
      server = ServerProxy("http://open-tran.eu/RPC2")
      for sugg in server.suggest("save as", lang):
          print sugg

이 방법을 이용하여 서버에서 추천 단어를 가져오는 간단한 .po 편집기를 제작할 예정입니다.
