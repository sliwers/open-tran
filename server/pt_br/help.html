Traduzindo Frases Simples

  1. Você pode adicionar um motor de pesquisas (dê uma olhada nas traduções sugeridas) a qualquer navegador web moderno.
  2. Não é segredo que o formulário de pesquisas usa um URL do seguinte formato: http://pt-br.open-tran.eu/suggest/search engine.

XML RPC

Você pode usar este serviço com qualquer programa capaz de chamar um procedimento remoto pelo protocolo XML RPC. Você deveria usar o método suggest, que aceita dois argumentos: uma frase para ser traduzida e o código do idioma para traduzir.

Em Python, você pode usar da seguinte forma: 

      from xmlrpclib import ServerProxy

      lang = "pt_BR" # replace with your language code
      server = ServerProxy("http://open-tran.eu/RPC2")
      for sugg in server.suggest("save as", lang):
          print sugg
