XML RPC

Môžete použiť túto službu s akýmkoľvek programom, ktorý je schopný volania vzdialenej procedúry pomocou XML RPC protokolu. Mali by ste použiť metódu suggest, ktorá prijíma dva argumenty: frázu na preloženie a umiestnenie kódu jazyka.

V programovacom jazyku Python, môžete túto službu použiť nasledujúcim spôsobom: 

      from xmlrpclib import ServerProxy

      lang = "pt_BR" # replace with your language code
      server = ServerProxy("http://open-tran.eu/RPC2")
      for sugg in server.suggest("save as", lang):
          print sugg

Budem sa snažiť čoskoro nahrať jednoduchý .po editor s ktorým sa budú dať vyzdvihnúť návrhy zo servera pomocou tohoto mechanizmu.
