XML RPC

To storitev lahko uporabite s katerimkoli programom, ki zmore poklicati oddaljen postopek preko protokola XML RPC. Uporabiti morate način suggest, ki sprejme dva argumenta: besedno zvezo, ki bi naj bila prevedena in kodo ciljnega jezika.

V Pythonu jo lahko uporabite na naslednji način: 

      from xmlrpclib import ServerProxy

      lang = "pt_BR" # zamenjajte s kodo ciljnega jezika
      server = ServerProxy("http://open-tran.eu/RPC2")
      for sugg in server.suggest("save as", lang):
          print sugg
