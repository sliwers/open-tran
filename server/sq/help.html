  1. Mund të shtoni një motor kërkimesh (shihni përkthimet e këshilluar) te cilido shfletues modern web.
  2. Nuk përbën ndonjë sekret që formulari i kërkimeve jep një URL nën formatin vijues: http://xx.open-tran.eu/suggest/search engine.

    Tani, duke dhënë një URL në formën togfjalësh për përkthim do të parashtrohet vetvetiu një kërkesë.

XML RPC

Këtë shërbim mund ta përdorni me cilindo program që është i aftë të thërrasë një procedurë në largësi përmes protokollit XML RPC. Do të duhej të përdornit metodën suggest, që pranon dy argumenta: një togfjalësh për t'u përkthyer dhe kodin e gjuhës për të cilën synohet përkthimi.

Në Python, mund ta përdorni në mënyrën e mëposhtme: 

      prej xmlrpclib import ServerProxy

      lang = "pt_BR" # zëvendësojeni me kodin e gjuhës suaj
      server = ServerProxy("http://open-tran.eu/RPC2")
      for sugg in server.suggest("save as", lang):
          print sugg

Së shpejti do të përpiqem të jap një përpunues të thjeshtë për kartelat .po, i cili do të jetë në gjendje të sjellë këshillime prej shërbyesash që përdorin këtë mekanizëm.
