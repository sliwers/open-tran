XML RPC

Ову услугу можете да користите са било којим програмом који може позивати даљинску процедуру преко XML RPC протокола. Требали бисте да користите метод suggest, који користи два аргумента: реченицу за превођење и кôд језика.

У Питону користите на следећи начин: 

      from xmlrpclib import ServerProxy

      lang = "sr" # замените са кодом свог језика
      server = ServerProxy("http://open-tran.eu/RPC2")
      for sugg in server.suggest("save as", lang):
          print sugg

Покушаћу да ускоро пошаљем једноставни .po уређивач који ће моћи повлачити предлоге са послужитеља користећи овај механизам.
