  2. Det är ingen hemlighet att sökrutan leder till en URL på följande format: http://xx.open-tran.eu/suggest/search engine.

XML RPC

Denna tjänst kan utnyttjas av program som klarar anropa en remote procedure via XML RPC-protokollet. Använd metoden suggest, som tar två parametrar: en fras som ska översättas och språkkoden för målspråket.

I Python, kan den anropas så här: 

      from xmlrpclib import ServerProxy

      lang = "pt_BR" # replace with your language code
      server = ServerProxy("http://open-tran.eu/RPC2")
      for sugg in server.suggest("save as", lang):
          print sugg

Jag hoppas kunna ladda upp en enkel .po-editor inom kort som kan hämta förslag från servern med den här funktionen.
