XML RPC

Цією службою можна користуватись з будь-якої програми, яка здатна викликати віддалену процедуру через протокол XML RPC. Потрібно вживати метод suggest, який приймає два аргументи: фразу, яку потрібно перекласти, і код мови, на яку проводиться переклад.

В Python це можна втілити наступним чином: 

      from xmlrpclib import ServerProxy

      lang = "uk" # змініть на код вашої мови
      server = ServerProxy("http://open-tran.eu/RPC2")
      for sugg in server.suggest("save as", lang):
          print sugg

Незабаром, я постараюсь вивантажити простий редактор .po, який буде спроможний отримувати пропозиції з сервера за допомогою цього механізму.
